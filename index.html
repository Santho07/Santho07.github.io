Get All Unique Timezone In Rails

Rails Timezone There is only 35 Unique TimeZone worldwide ( According to Rails Timezone Class ). We can get a list of all the unique time happening worldwide by running the following snippet from rails console and see it ourselves. :smile: ActiveSupport::TimeZone.all.map(&:name).map do |zone| Time.now.in_time_zone(zone).strftime('%F %T %:z') }.uniq end To get the full list of 150 timezones ActiveSupport::TimeZone.all.map(&:name).map do |zone| zone + ' : ' + Time.now.in_time_zone(zone).strftime('%F %T %:z') } end Thank you for reading!...
